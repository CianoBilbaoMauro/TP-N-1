*¿Que es HTML?

Es un lenguaje de marcado que nos permite
indicar la estructura de nuestro documento mediante etiquetas.
Este lenguaje nos ofrece una gran adaptabilidad, una estructuración lógica
y es fácil de interpretar tanto por humanos como por máquinas.

*¿Que es CSS (Cascading Style Sheets)?

CSS son las siglas en inglés para "hojas de estilo en cascada".
Básicamente, es un lenguaje que maneja el diseño
y presentación de las páginas web, es decir,
cómo lucen cuando un usuario las visita
